Coffee++

Coffee++ is a little language that compiles into C++. It has been created to have something similar to CoffeeScript for C++. Currently Coffee++ is in a pre-alpha state and not at all usable or final. Checkout the source on Github to get involved.

The golden rule of Coffee++ is: "It's just C++". The code compiles one-to-one into the equivalent C++, and there is no interpretation at runtime. You can use any existing C++ library seamlessly from Coffee++ (and vice-versa).

Overview

source file Test.cf++

include iostream

class Test:
	public new():
		bar = 123
	public void foo():
		std::cout << "Coffee++!\n"
	private int bar

int main():
	int a = 5
	if a != 7:
		Test test
		test.foo()

compiled Test.hpp

#pragma once

class Test {
public:
	Test();
	void foo();
private:
	int bar;
};

int main();

compiled Test.cpp

#include "test.h"

#include <iostream>

Test::Test() : bar(123) {
}

void Test::foo() {
	std::cout << "Coffee++!\n";
}

int main() {
	int a = 5;
	if (a != 7) {
		Test test;
		test.foo();
	}
}

Language Reference

Include

To include a library you can use the include statement:

Example.cf++

include iostream
include boost/array.hpp header

Example.hpp

#include <boost/array.hpp>

Example.cpp

#include "Example.hpp"

#include <iostream>

If you want to include a local file use import. Appending forward will also forward declare the class with the same name as the file:

Example.cf++

import legacy.h
import Foo
import Bar forward
import Test header

Example.hpp

#include "Test.hpp"

class Bar;

Example.cpp

#include "Example.hpp"

#include "legacy.h"
#include "Foo.hpp"
#include "Bar.hpp"
